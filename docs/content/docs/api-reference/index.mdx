## Namespace Summary

### Zap

Core serialization types:

```csharp
using Zap;

// Message building
var message = MessageBuilder.Create();
var root = message.BuildRoot<MyStruct.WRITER>();

// Deserialization
var state = DeserializerState.CreateRoot(segments);
var reader = state.RequireStruct().Read<MyStruct.READER>();

// I/O
Framing.Write(stream, message.Frame);
var segments = Framing.ReadSegments(stream);
```

### Zap.Rpc

RPC types:

```csharp
using Zap.Rpc;

// Client
using var client = new TcpRpcClient("localhost", 5000);
await client.WhenConnected;
var service = client.GetMain<IMyService>();

// Server
using var server = new TcpRpcServer();
server.Main = new MyServiceImpl();
server.StartAccepting(IPAddress.Any, 5000);
```

### Zap.FrameTracing

Debugging types:

```csharp
using Zap.FrameTracing;

public class MyTracer : IFrameTracer
{
    public void TraceFrame(WireFrame frame, bool isIncoming) { }
}
```
