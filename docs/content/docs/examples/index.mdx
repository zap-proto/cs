## Running Examples

Each example includes:

1. ZAP schema definition
2. Server implementation
3. Client usage
4. Complete working code

To run an example:

```bash
# Create new project
dotnet new console -n ExampleProject
cd ExampleProject

# Add packages
dotnet add package Capnp.Net.Runtime
dotnet add package CapnpC.CSharp.MsBuild.Generation

# Add schema file and code
# Build and run
dotnet run
```

## Common Patterns

### Request-Response

Simple RPC calls with immediate responses.

```csharp
var result = await service.Calculate(input);
```

### Callbacks

Server calls back to client-provided capability.

```csharp
await service.Subscribe(new MyHandler());
```

### Streaming

Continuous data flow using callback sequences.

```csharp
await service.StreamData(new DataHandler());
await dataHandler.WaitForCompletion();
```

### Promise Pipelining

Chain calls without waiting for intermediate results.

```csharp
var finalResult = await service.GetFactory()
    .CreateProcessor()
    .Process(data);
```
